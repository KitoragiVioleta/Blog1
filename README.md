# ✍️ Мій блог

Простий блог з CRUD-функціоналом на Ruby on Rails.

## Технології
- Ruby on Rails 8.1
- Vanilla JavaScript
- localStorage для збереження постів

## Як запустити
1. `bundle install`
2. `rails server`
3. Відкрий http://localhost:3000

## Функціонал
- Створення, редагування, видалення постів
- Фільтрація за категоріями
- Пошук по заголовку та тексту
- Теги
- Дизайн згенеровано за допомогою Gemini AI
